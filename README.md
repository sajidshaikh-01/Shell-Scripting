# Shell Scripting - Day 1 Summary

## ðŸ”¹ What is Shell Scripting?
Shell scripting is a way to automate Linux/Unix commands by writing them inside a file (script).  
It is used in DevOps for automation, CI/CD, and system management.

---

## ðŸ”¹ Variables
Variables are used to store and reuse data values in a script.  
Access variable values using `$variable_name`.

---

## ðŸ”¹ Arrays
Arrays are used to store multiple values in a single variable.  
Each element can be accessed using its index.  

---

## ðŸ”¹ Strings
A string is a sequence of characters used to represent text data.  
We can perform operations like length check, substring, or replacement.

---

## ðŸ”¹ User Interaction
Allows users to provide input while the script is running using the `read` command.  

---

## ðŸ”¹ Arithmetic Operations
Used to perform mathematical calculations such as addition, subtraction, multiplication, and division.  

---

## ðŸ”¹ Conditions
Used to make decisions in a script based on certain conditions (true or false).  

---

## ðŸ”¹ if Statement
Executes a block of code if a specific condition is true.  

---

## ðŸ”¹ if-else Statement
Executes one block if the condition is true, otherwise executes another block.  

---

## ðŸ”¹ elif Statement
Used to check multiple conditions sequentially.  

---

## ðŸ”¹ case Statement
Used for multiple condition handling in a clean and readable format.
